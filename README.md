# Rocket League MVP

## Overview
Rocket League MVP is a multiplayer vehicular soccer video game that combines physics-based gameplay with competitive sports mechanics. Players control rocket-powered cars to hit a ball into the opposing team's goal to score points.

## Features
- **Multiplayer Modes**: Play with friends in 2v2, 3v3, or 4v4 matches.
- **Customization**: Customize your vehicle's appearance with various skins and accessories.
- **Power-ups**: Use special abilities to gain an advantage in matches.
- **Competitive Rank**: Climb the ranks in competitive modes.
- **Seasonal Events**: Participate in time-limited events for unique rewards.

## Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Malacos/RocketLeague.git
   cd RocketLeague
   ```
2. **Ensure you have the required dependencies**:
   - Unity 2020.3 or higher
   - .NET Framework 4.7.1 or higher

3. **Open the project in Unity**:
   - Open Unity Hub.
   - Click on "Add", and select the cloned `RocketLeague` folder.
   - Open the project.

4. **Build the project**:
   - In Unity, navigate to `File > Build Settings`.
   - Select your target platform and click on `Build`.

## Usage
- Launch the game from the built executable.
- Use the arrow keys or WASD to control your vehicle.
- The goal is to score more points than the opposing team within the match time.
