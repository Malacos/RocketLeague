# Installation and Configuration Guide for RocketLeague

## Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Malacos/RocketLeague.git
   ```

2. **Navigate into the project directory**:
   ```bash
   cd RocketLeague
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

## Configuration
1. **Open the configuration file** (usually located in the `config` directory).

2. **Edit the necessary fields** to suit your setup. The following fields need to be configured:
   - `API_KEY`: Your API key for accessing external services.
   - `DATABASE_URL`: The URL for connecting to your database.

3. **Save the changes** and make sure to restart the application to apply the configuration changes.
